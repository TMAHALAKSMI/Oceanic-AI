# 🌊 OceanWatch AI — Ocean Waste Detection System

AI-powered detection and classification of marine debris from drone/satellite imagery.

## Features

| Feature | Description |
|---|---|
| **AI Detection** | YOLOv8 (preferred) or rule-based colour/contour fallback |
| **8 Waste Classes** | Plastic Bottle, Plastic Bag, Fishing Net, Metal, Organic, Foam, Rope, Unidentified |
| **Bounding Boxes** | Annotated output with label + confidence overlay |
| **Thermal Simulation** | JET colormap heatmap to simulate IR/thermal imaging |
| **Comparison Mode** | Side-by-side clean vs polluted image analysis |
| **Dashboard** | History, KPI cards, doughnut chart |
| **SQLite Storage** | All results persisted locally |

## Quick Start

### 1. Prerequisites

```bash
Python 3.10+
pip
(Optional) CUDA-capable GPU for faster YOLO inference
```

### 2. Install backend dependencies

```bash
cd backend
pip install -r requirements.txt
```

> **Minimal install** (no YOLO, uses built-in rule-based detector):
> ```bash
> pip install fastapi uvicorn python-multipart opencv-python-headless numpy
> ```

### 3. Prepare a demo dataset (optional)

```bash
python prepare_dataset.py --source demo --n 300 --output dataset/
```

### 4. Train the model (optional — skip to use rule-based fallback)

```bash
python train_model.py --data dataset/data.yaml --epochs 50 --batch 16
# Best weights auto-copied to backend/models/ocean_waste_yolov8.pt
```

### 5. Start the API server

```bash
cd backend
uvicorn main:app --host 0.0.0.0 --port 8000 --reload
```

API docs: http://localhost:8000/docs

### 6. Open the frontend

```bash
# Open in browser directly (no build step):
open frontend/templates/index.html

# OR serve with Python:
cd frontend
python -m http.server 3000
# Visit http://localhost:3000/templates/index.html
```

## Detection Model

### Priority order:
1. **Custom YOLOv8** — `backend/models/ocean_waste_yolov8.pt` (if exists)
2. **YOLOv8n pretrained** — downloads on first run, remaps COCO classes to waste categories
3. **Rule-based fallback** — colour/contour heuristics using OpenCV (zero ML dependencies)

### Waste classes:
```
0  Plastic Bottle
1  Plastic Bag
2  Fishing Net
3  Metal Debris
4  Organic Waste
5  Foam/Styrofoam
6  Rope/Twine
7  Unidentified Debris
```
